
# 🎬 ভিডিও স্ক্রিপ্ট: ব্যাশ স্ক্রিপ্টিং (দিন - ০২)
## বিষয়: ফাইল সিস্টেম নেভিগেশন এবং ফাইল হ্যান্ডলিং
### **দৃশ্য ১: হুক ও পরিচিতি (০০:০০ - ০১:০০)**
**(স্ক্রিনে: একটি ল্যাপটপ স্ক্রিন যেখানে টার্মিনাল দ্রুত কাজ করছে)**
**কথা:**
"আসসালামু আলাইকুম! ব্যাশ স্ক্রিপ্টিং সিরিজের দ্বিতীয় পর্বে আপনাদের স্বাগতম। গত পর্বে আমরা ব্যাশ সেটআপ করেছি এবং echo কমান্ডের ম্যাজিক দেখেছি। কিন্তু আজ আমরা আরও গভীরে যাবো।
আপনি যদি লিনাক্স বা সার্ভার নিয়ে কাজ করতে চান, তবে আপনাকে ফাইল এবং ফোল্ডারের সাথে কথা বলা শিখতে হবে। মাউস ছাড়া কীভাবে এক ফোল্ডার থেকে অন্য ফোল্ডারে যাবেন, ফাইল কপি করবেন বা ডিলিট করবেন—আজকের ভিডিওতে আমরা সেটাই শিখবো। আজকের ভিডিওটি একটু বড় হতে পারে, কিন্তু এটিই আপনার লিনাক্স জার্নির সবচেয়ে গুরুত্বপূর্ণ ভিত্তি।"
### **দৃশ্য ২: আমি এখন কোথায়? (pwd & ls) (০১:০০ - ০৩:০০)**
**(স্ক্রিনে: টার্মিনাল ওপেন করা এবং টাইপ করা হচ্ছে)**
**কথা:**
"টার্মিনাল ওপেন করার পর আমাদের প্রথম প্রশ্ন থাকে—আমি এখন কোথায় আছি? এটি জানতে টাইপ করুন pwd।"
**(টার্মিনাল ডেমো):**
```bash
pwd
# আউটপুট দেখাবে আপনার বর্তমান লোকেশন, যেমন: /home/user/Desktop

```
**কথা:**
"এখন এই লোকেশনে কী কী ফাইল বা ফোল্ডার আছে তা দেখতে আমরা ব্যবহার করি ls বা List কমান্ড।"
**(টার্মিনালে ডেমো):**
```bash
ls        # শুধু ফাইলের নাম দেখতে
ls -l     # ডিটেইলস (সাইজ, সময়) দেখতে
ls -a     # হিডেন বা লুকানো ফাইল দেখতে (যেগুলোর শুরুতে ডট আছে)

```
### **দৃশ্য ৩: ফোল্ডারে যাতায়াত (cd) (০৩:০০ - ০৫:০০)**
**(স্ক্রিনে: ফোল্ডার ডিরেক্টরি ট্রি এর একটি এনিমেশন)**
**কথা:**
"এখন চলুন এক ফোল্ডার থেকে অন্য ফোল্ডারে যাই। এর জন্য কমান্ড হলো cd বা Change Directory।"
**(টার্মিনালে ডেমো):**
```bash
cd Documents    # 'Documents' ফোল্ডারের ভেতরে ঢোকার জন্য
cd ..           # এক ধাপ পেছনে বা আগের ফোল্ডারে ফেরত আসার জন্য
cd ~            # সরাসরি নিজের হোম ডিরেক্টরিতে চলে যাওয়ার জন্য

```
### **দৃশ্য ৪: ফাইল ও ফোল্ডার তৈরি (mkdir & touch) (০৫:০০ - ০৭:৩০)**
**(স্ক্রিনে: দ্রুত কিছু ফোল্ডার তৈরি হতে দেখা যাচ্ছে)**
**কথা:**
"এখন আমরা কিছু তৈরি করবো। নতুন ফোল্ডার বানাতে ব্যবহার করুন mkdir এবং নতুন খালি ফাইল বানাতে touch।"
**(টার্মিনালে ডেমো):**
```bash
mkdir MyTutorial       # একটি নতুন ফোল্ডার
cd MyTutorial          # ফোল্ডারের ভেতরে ঢুকলাম
touch script.sh index.js  # একসাথে দুটি ফাইল তৈরি করলাম

```
### **দৃশ্য ৫: কপি এবং মুভ (cp & mv) (০৭:৩০ - ১০:০০)**
**(স্ক্রিনে: স্ক্রিন ভাগ করে বামে সোর্স এবং ডানে ডেস্টিনেশন দেখানো হচ্ছে)**
**কথা:**
"ফাইল ম্যানেজমেন্টের সবচেয়ে জরুরি অংশ হলো কপি এবং মুভ। কোনো ফাইল কপি করতে লিখুন cp।"
**(টার্মিনালে ডেমো):**
```bash
cp script.sh backup_script.sh   # ফাইলের একটি কপি তৈরি হলো
cp -r MyTutorial MyTutorial_Backup # পুরো ফোল্ডার কপি করতে -r ব্যবহার করুন

```
**কথা:**
"আর ফাইল সরাতে বা নাম পরিবর্তন (Rename) করতে ব্যবহার করুন mv।"
**(টার্মিনালে ডেমো):**
```bash
mv script.sh Documents/      # ফাইলটি অন্য ফোল্ডারে চলে গেল
mv index.js main.js          # index.js এর নাম বদলে main.js হয়ে গেল

```
### **দৃশ্য ৬: ফাইল দেখা এবং ডিলিট করা (cat & rm) (১০:০০ - ১২:৩০)**
**(স্ক্রিনে: ফাইলে কিছু টেক্সট লিখে সেটি টার্মিনালে দেখানো হচ্ছে)**
**কথা:**
"ফাইল তো বানালাম, কিন্তু এর ভেতর কী আছে তা টার্মিনালেই দেখতে চাইলে টাইপ করুন cat।"
**(টার্মিনালে ডেমো):**
```bash
cat main.js  # ফাইলের ভেতর যা আছে তা স্ক্রিনে দেখাবে

```
**কথা:**
"সবশেষে, কোনো ফাইল ডিলিট করতে চাইলে কমান্ড হলো rm (Remove)। সাবধান! এটি ব্যবহার করলে ফাইল সরাসরি ডিলিট হয়ে যায়, ট্র্যাশে যায় না।"
**(টার্মিনালে ডেমো):**
```bash
rm main.js             # ফাইল ডিলিট
rm -rf MyTutorial      # কোনো ফোল্ডার এবং তার সব ফাইল জোরপূর্বক ডিলিট করা

```
### **দৃশ্য ৭: প্রো-টিপস এবং শর্টকাট (১২:৩০ - ১৪:০০)**
**(স্ক্রিনে: কিবোর্ড শর্টকাটগুলো হাইলাইট হচ্ছে)**
**কথা:**
"আপনার কাজকে আরও সহজ করতে তিনটি শর্টকাট মনে রাখুন:
১. **Tab Key:** অর্ধেক নাম লিখে ট্যাব চাপুন, ব্যাশ অটো-কমপ্লিট করে দেবে।
২. **Up Arrow:** আগের দেওয়া কমান্ডগুলো ফিরে পেতে কিবোর্ডের উপরের অ্যারো চাপুন।
৩. **Clear:** টার্মিনাল হিজিবিজি হয়ে গেলে clear লিখে স্ক্রিন পরিষ্কার করে ফেলুন।"
### **দৃশ্য ৮: হোম টাস্ক ও বিদায় (১৪:০০ - শেষ)**
**(স্ক্রিনে: আজকের সব কমান্ডের একটি চিট-শিট বা চার্ট)**
**কথা:**
"আজকের জন্য আপনাদের একটি ছোট্ট কাজ দিচ্ছি:
১. আপনার নামে একটি ফোল্ডার বানান।
২. তার ভেতরে test.txt নামে একটি ফাইল বানান।
৩. ফাইলটির নাম পরিবর্তন করে done.txt করুন।
৪. সবশেষে ফোল্ডারটি ডিলিট করে দিন।
ব্যাস! এইটুকু করতে পারলেই আপনি আজকের লেসন বুঝে গেছেন। আগামী পর্বে আমরা দেখবো কীভাবে এই কমান্ডগুলো দিয়ে একটি প্রপার 'স্ক্রিপ্ট ফাইল' বানানো যায়। ভিডিওটি ভালো লাগলে লাইক দিন এবং আপনার বন্ধুদের সাথে শেয়ার করুন। আল্লাহ হাফেজ!"
